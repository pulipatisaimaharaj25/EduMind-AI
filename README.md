# EduMind AI 🚀 — Your Personal Study Assistant

**EduMind AI** is an innovative educational application designed to empower rural Indian students with high-quality, personalized academic support. It addresses the educational divide by combining **Google Gemini AI** and local **SQLite/Room caching** to provide interactive learning support in both **simple English** and **Telugu**.

This project was built for submission to the **Education & Learning** hackathon track, targeting the 250+ million school students in India who lack access to premium personalized tutoring.

---

## 📖 Problem Statement
Rural students in India face unique educational hurdles:
1. **Lack of Personal Tutors:** Private tutoring is expensive, and school classrooms are heavily crowded.
2. **Language Barrier:** Most digital education products are written in complex academic English, making them inaccessible to students from regional language mediums (like Telugu).
3. **Connectivity & Device Access:** Many students rely on low-cost family mobile devices and unstable network connections.

### The EduMind AI Solution
EduMind AI provides a **completely free**, **highly interactive**, and **mobile-first** assistant:
* **Bilingual Support:** Instantly switches between simple, easy-to-read English and native **Telugu** at the tap of a button.
* **On-Device Offline Tracking:** Persists registration data, historic test scores, and day-by-day study schedules in a local database (SQLite/Room), keeping the app stable and robust.
* **High Readability:** Utilizes large, vibrant tactile chips, Material 3 spacing, and high-contrast styling tailored for small mobile screens.

---

## ✨ Core Features

### 1. 💬 AI Doubt Solver (Bilingual Chat)
* Students can select standard subjects (**Mathematics, Science & Tech, English Grammar, Social Studies, Computer Literacy**) and type in any doubt.
* The Google Gemini API translates the question and responds in **extremely simple, step-by-step lines** with local Indian context examples.
* Features an instant toggle to receive explanations in **Telugu (తెలుగు)**!

### 2. 📝 Personalized Quiz Generator
* AI generates custom **5 Multiple-Choice Questions (MCQs)** based on the student's selected subject, difficulty, and class (from 6th to 12th standard).
* Provides real-time visual feedback: color-coding selections immediately in **Green (Correct)** or **Red (Incorrect)**.
* Shows custom **AI explanations** for each question, helping the student learn from their mistakes.

### 3. 📅 Interactive Study Planner
* Students input their exam name, target timeframe, and subjects.
* Gemini compiles a customized, reasonable day-by-day schedule shown in an elegant calendar checklist view.
* Students can check off topics as they complete them; the progress is automatically persisted back to SQLite!

### 4. 📊 Student Workspace Dashboard
* **Dynamic AI Quotes:** Greets the student with daily motivational quotes generated by Gemini based on their name.
* **Progress Tracking:** Shows high-contrast progression gauges for each major subject area.
* **Score History Card:** Displays the last 5 quiz scores, indicating passed modules and areas to review.
* **Today's Priorities:** Highlights today's active study plan tasks with working checkbox controls.

### 5. 🚀 Landing Onboarding Experience
* A professional, welcoming greeting page introducing the student to all features.
* Responsive, high-touch name and class standard configuration panel that creates their profile with zero password complexity.

## 🛠️ Technical Stack
* **Language:** Kotlin (100% type-safe)
* **UI Framework:** Jetpack Compose (Material Design 3)
* **AI Core:** Google Gemini REST API (`gemini-3.5-flash` model)
* **Database / Cache:** Room Database (SQLite engine with KSP)
* **Network & Serialization:** Retrofit 2 + Moshi (JSON SerDe)
* **Concurrency:** Kotlin Coroutines & Flow

## 🚀 How to Run Locally

### Prerequisites
* Android Studio (Koala or newer)
* Android SDK 34+
* An Android device or emulator running API level 24 (Android 7.0) or higher

### Step-by-Step Setup

1. **Clone the Project:**
   ```bash
   git clone https://github.com/your-username/edumind-ai.git
   cd edumind-ai
   ```

2. **Configure the Gemini API Key:**
   EduMind AI securely loads its API keys using the **Secrets Gradle Plugin**.
   * Create a file named `.env` in the root directory of the project.
   * Add your Google Gemini API key:
     ```env
     GEMINI_API_KEY=AIzaSyYourActualGoogleGeminiKeyHere
     ```
   * *Note: Do NOT commit this `.env` file to public repositories!*

3. **Open in Android Studio:**
   * Launch Android Studio and select **Open**.
   * Select the root `edumind-ai` folder and let Gradle index and sync the dependencies.

4. **Build and Run:**
   * Connect your physical Android phone (with USB debugging enabled) or start a Virtual Device (Emulator).
   * Click the **Run** green play button in Android Studio, or execute in the terminal:
     ```bash
     gradle assembleDebug
     ```

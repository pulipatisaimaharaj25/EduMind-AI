# EduMind AI 🚀 — Your Personal Study Assistant

AI-powered personalized study assistant for rural Indian 
students — built with Python Flask, SQLite, and Gemini AI.

## 🎯 Hackathon Track
Education & Learning — CodeFusion 2026

## 📖 Problem Statement
Rural Indian students lack access to personalized tutoring,
face language barriers, and have limited resources.

## ✨ Features
- 💬 AI Doubt Solver (English + Telugu)
- 📝 Personalized Quiz Generator (6th-12th class)
- 📅 Study Planner with day-by-day schedule
- 📊 Student Dashboard with progress tracking

## 🛠️ Tech Stack
- Backend: Python Flask
- Database: SQLite
- AI: Google Gemini API
- Frontend: HTML, CSS, JavaScript

## 🚀 How to Run
pip install -r requirements.txt
python app.py
